 * Logging in FastAPI
 * Async databases
 * Logging in Betterstack
 * User Authentication
 * SqlAlchemy relationships
 * User Email confirmation
 * File Uploads
 * Background tasks
 * Image generation using OpenAi based on post body

 * Github Actions for linting and testing
